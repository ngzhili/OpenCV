# COCO Viewer Application
A python Tkinter GUI Application to view coco annotations on your local machine easily. Support Windows only.

# Description
Two Main Functionalities:
1. Create a masks of bounding box and segmentation object classes on your image dataset (supports only .jpg, .jpeg, .png, .JPG, .JPEG , .PNG image file types) with annotations in coco.json format.
2. Image Viewer to view your downloaded masked images on a new window.

# Installation
No external library packages needed to be installed.
## Steps:
1. Download and install coco viewer application folder on your local machine. 
2. Extract all files in .zip. 
3. Double click on coco_viewer.exe application. Application should being launching.
